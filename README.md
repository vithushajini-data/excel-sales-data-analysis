#  📊 Excel Sales Data Analysis Project

I built a sales analysis project in Microsoft Excel using a sample dataset to practice real-world data analysis techniques.

## 🛠️ Tools Used
- Microsoft Excel
- Functions: SUM, IF, COUNTIF, SUMIF, VLOOKUP, XLOOKUP

## 📈 What I did
- Calculated total sales using Excel formulas  
- Classified performance (High / Low) using IF function  
- Analyzed sales by region using SUMIF  
- Retrieved product details using lookup functions  
- Structured and summarized raw data into meaningful insights  

## 💡 Key Learning
This project helped me understand how Excel is used in real business environments for reporting, analysis, and decision making.
